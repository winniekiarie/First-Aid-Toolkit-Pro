# First-Aid-Toolkit-Pro# 🚑 First Aid Toolkit Pro

## 📍 Overview

First Aid Toolkit Pro is a beginner-friendly web application that provides quick and simple first aid instructions for common emergencies. It is designed to demonstrate how basic web technologies can be used to build practical, real-world solutions.

This project was developed as part of a capstone assignment to explore learning a new technology using AI-assisted prompts.

---

## 🎯 Project Objective

* Learn and apply HTML, CSS, and JavaScript
* Build a simple interactive web application
* Demonstrate real-world problem solving using technology
* Document the learning process using AI prompts

---

## ✨ Features

* 🩹 First aid guidance for:

  * Burns
  * Cuts
  * Choking
  * Fractures
  * Bleeding

* 🔍 Search functionality to quickly find topics

* 🔊 Text-to-speech (Read Aloud feature)

* 🚨 Emergency alert button

* 📱 Responsive and clean user interface

---

## 🛠️ Technologies Used

* HTML (Structure)
* CSS (Styling)
* JavaScript (Interactivity)

## 💻 How to Run the Project

### Option 1: Run Locally

1. Download or clone this repository
2. Open the project folder
3. Double-click on `index.html`
4. The app will open in your browser

---

### Option 2: Using VS Code (Recommended)

1. Open the folder in Visual Studio Code
2. Right-click `index.html`
3. Select **"Open with Live Server"** (if installed)

## ⚠️ Common Issues & Fixes

### Issue: Buttons not working

✔ Ensure `script.js` is properly linked in HTML

### Issue: Styles not applying

✔ Check if `style.css` is correctly linked

### Issue: No sound from Read Aloud

✔ Ensure your browser allows audio playback
